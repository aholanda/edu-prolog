== Introdução

Prolog é uma linguagem de propósito geral projetada por Alain
Colmerauer no início da década de 1970. Prolog é uma linguagem
declarativa, e as expressões lógicas são representadas em termos de
fatos e regras.

Prolog tem sido usada para desenvolvimento de sistemas em diversas
áreas, dentre elas: inteligência artificial, sistemas especialistas,
banco de dados relacional, processamento de linguagem natural e
sistemas de prova automatizada de teoremas.
